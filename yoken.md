# アプリ開発要件定義書：Lofi Kappa (Hydration & Evolution)

## 1. プロジェクト概要
- **アプリ名**: Lofi Kappa（ロファイ・カッパ）
- **コンセプト**: 日々の水分補給で不思議な「卵」を潤し、あなただけのヘンテコで愛らしい「かっぱ」へと孵化・進化させてアルバムに集めていく、Cozyな放置・育成習慣化アプリ。
- **主要技術**: iOS (SwiftUI), WidgetKit.
- **ビジュアルスタイル**: 温かみのある手描き風（doodle style / 落書き風）のゆるいイラスト。アナログ手帖（ジャーナル）のような優しく落ち着いた世界観。

## 2. ユーザー体験 (Core Loop：卵からかっぱへの進化)
水分補給の進捗状況（ml）に応じて、1日の間に卵から最終形態（かっぱ）までダイナミックに変化します。

### 目標設定（mlダイレクト管理）
- **女性モード**: 1日の目標 2500ml
- **男性モード**: 1日の目標 3500ml

### 段階的な進化（5ステージの固定画像切り替え・座標計算なし）
- **Stage 1: 卵期 (0% 〜 20%未満)**
  - 最初はシンプルな「水の卵」のイラスト。
  - 女性: 0ml ≤ 水分量 < 500ml
  - 男性: 0ml ≤ 水分量 < 700ml
- **Stage 2: ひび割れ期 (20% 〜 40%未満)**
  - 卵の殻にひびが入り、中からほんのり水が漏れ、揺れ動く様子。
  - 女性: 500ml ≤ 水分量 < 1000ml
  - 男性: 700ml ≤ 水分量 < 1400ml
- **Stage 3: 幼生期 (40% 〜 60%未満)**
  - 卵の殻を破って、しっぽの生えた「おたまじゃくし風のかっぱ」が生まれます。
  - 女性: 1000ml ≤ 水分量 < 1500ml
  - 男性: 1400ml ≤ 水分量 < 2100ml
- **Stage 4: 成長期 (60% 〜 80%未満)**
  - しっぽが短くなり、ちいさな手足が生えてよちよちと動き始める状態。
  - 女性: 1500ml ≤ 水分量 < 2000ml
  - 男性: 2100ml ≤ 水分量 < 2800ml
- **Stage 5: 成体期（完成）(80% 〜 100%以上)**
  - 立派にお皿がのったお披露目状態（デフォルトの基本かっぱ）。
  - 女性: 2000ml ≤ 水分量
  - 男性: 2800ml ≤ 水分量

### ゴール
1日の目標（女性 2500ml / 男性 3500ml）に100%到達すると、その瞬間に完全ランダム（フラットな等確率）で決定された「ユニークなかっぱ」が解放（画像が差し替え）されます。
解放されたかっぱは「思い出アルバム（図鑑）」へコレクション。翌日はまた新しい卵がセットされます。

## 3. 水分記録の制御ロジック（不正・過剰摂取防止）

### A. 吸収インターバル (Cooldown)
- **仕様**: 記録後、15分間は次の記録を受け付けない。
- **UI**: ウィジェットのボタンが「満水状態（Satisfied）」または「吸水中（Absorbing）」になり、ロックされる。

### B. 過剰摂取チェック & お皿の溢れペナルティ (Overflow Penalty)
- **チェック条件**: 1時間あたりの摂取量が 1000ml（1.0L）を超えた場合。
- **ペナルティ**: 「お皿の限界（水溢れ）」と判断し、「今日1日で飲んだ水分量」がすべてリセット（0ml・卵の状態）される。
- **演出**: ウィジェット内のお皿から水が激しく溢れ出し、かっぱ（または卵）が目を回してずぶ濡れになる「FLOOD（洪水）」演出が発生。

## 5. 全体のデザイン要件 (Design Guidelines)
アプリ全体は、デジタル機器であることを忘れさせるような「1冊の古いアナログ手帖（ノートブック）」を手元で開いているかのようなCozyな質感で統一します。

### A. テーマとマテリアル質感
- **ライトスキン（ペーパー・セピア）**:
  真っ白な画面は避け、温かみのある古いザラついた紙（羊皮紙・クラフト紙）を模したテクスチャや、極めて薄いベージュ・セピア色をベース背景色にします。枠線や区切り線には、定規を使わずに万年筆やミリペンで引いたような、微妙に歪みのある「手書き調のグレーの二本線」などを採用します。
- **ダークスキン（ミッドナイト・ウォーム）**:
  冷たい青黒さ（黒100%）ではなく、少し温かみのある「セピア寄りのダークチャコール（灰褐色）」をベースにし、暖色の常夜灯に照らされているような安心感を演出します。

### B. カラーパレット (Lofi Color Palette)
手描きイラストやUIで共通して使用する、彩度を少し落とした目に優しいペールトーンです。
- **Base Background**: `#FAF6F0` (暖かみのある紙の白) / `#1C1917` (ダークブラウンチャコール)
- **Paper Card Layer**: `#FFFDFB` (カードやポップアップの紙色) / `#292524` (ダーク時のカード)
- **Ink/Text**: `#3F3F46` (チャコールグレー・万年筆のインク風) / `#E4E4E7` (ホワイトウォッシュ)
- **Kappa Green**: `#A7F3D0` (かっぱの基本色・淡いミントグリーン) / `#34D399` (アクセントグリーン)
- **Hydration Blue**: `#93C5FD` (潤いの水色・淡いパステルブルー) / `#60A5FA` (お皿の水、プログレスバー)
- **Penalty Rust**: `#FCA5A5` (警告・お皿が溢れた時のサビ茶・サーモンピンク)

### C. タイポグラフィ (Typography)
システムの画一的なサンセリフフォントではなく、優しく角の丸いフォントや手書き風フォントをメインに据えます。iOSで実装する際は、標準フォントを使用する場合でも丸みのある `Font.system(.body, design: .rounded)` を指定し、手帖に鉛筆で書いたような「柔らかい筆致」を感じさせます。

### D. 時間帯ライティング (Time Lighting)
メインのホーム画面の背景イラストは、スマホの内蔵時計（時間帯）に連動してイラストの背後から差し込む「明かりの色」がゆっくりと変化します。
- **朝 (6:00 - 10:00)**: 新鮮なペールイエローと爽やかな薄いブルーのグラデーション。
- **昼 (10:00 - 16:00)**: ぽかぽかとした暖かなホワイトゴールド。
- **夕方 (16:00 - 19:00)**: ノスタルジーをそそるアンバー（琥珀色）と夕焼けピンク。
- **夜・深夜 (19:00 - 6:00)**: ランプがぽつんと灯る、静かなウォームオレンジとミッドナイトブルー。

## 6. ウィジェット設計
- **インタラクティブ**:
  - **クイック記録ボタン**: 1タップで、事前に設定したマイコップの容量（例: 250ml や 350ml）を直接mlデータとして記録。
  - **Progress Bar**: 100%（目標ml）までの進み具合を、現在の合計ml数値とともに可視化。
- **状態変化**:
  通常（まったり）、インターバル（ロック）、お皿の溢れ（洪水・ペナルティ）の3モード。

## 7. 画面設計 (Screen Design)
アプリ本体は下部タブナビゲーション（Tab View）を採用し、Lofiな優しい手描きイラストが主役になるよう、アナログ手帖風の落ち着いた配色（ライトまたはダークのペーパースキンテーマ）で統一します。

### ① ホーム画面 (Home View)
1日の中心となる、かっぱの育成・観察・記録を行うメイン画面。
- **ビジュアルエリア（上部〜中央）**:
  - 部屋の和室・または縁側の温かみのある手描き背景。時間帯に合わせて背景の明かりが変化（朝・昼・夜・深夜）。
  - 画面中央に、現在の進化ステージに合わせたゆるいイラスト（卵・卵ひび割れ・おたまじゃくし・手足生え・成体）を大きく配置（ウィジェットで表示するものと同じ画像）。
  - 呼吸をするように、ゆっくり上下に伸縮（Squash & Stretch）するアニメーションをループ。
- **進捗インジケーター**:
  - イラストの下に、進捗を示す手描き風のプログレスバー。
  - 表示テキスト: 「本日の水分量 / 目標ml（例: 1250 / 2500 ml）」と「進化までに必要な水分量 / 目標ml（例: 1250 / 2500 ml）」
- **アクションボタンエリア（下部）**:
  - **「マイコップ」ボタン（2つのクイックカスタム枠）**: 「コップ（例: 250ml）」と「ボトル（例: 500ml）」のクイック加算。
  - **コールドダウン表示**: 15分インターバル中はボタンが「吸水中（お皿を満たし中…）」に変わりグレーアウト、タイマーのカウントダウンが表示される。
  - **お皿溢れ警告灯**: 1時間の摂取量が 1000ml に近づくと、アナログ風の警告灯が点滅。溢れた瞬間に「FLOOD（全リセット）」アニメーションが割り込む。

### ② コレクション画面 (Album View)
これまでに解放したユニークなかっぱたちを鑑賞する図鑑画面。
- **グリッドリスト**:
  - 3 × N の木製シェルフ（手描き風の棚）風のレイヤー。
  - 未解放のかっぱは、シルエットと「？」マークで表示される。
- **詳細ポップアップ (Detail Card)**:
  - グリッド内の解放済みかっぱをタップすると、レトロな紙のカード（御札風、または手描きの手帖風）のデザインがポップアップ。
  - 拡大イラスト、タイトル、ゆるい説明テキスト、初めて解放された日付が表示される。
  - **「ストーリーズにシェア」ボタン**: このカードをそのままインスタ等の背景付き画像として書き出し。

### ③ ログ画面 (Log View)
自分の日々の水分補給の努力がどれだけ積み重なっているかを確認する、シンプルな振り返り画面。
- **週間水分グラフ**:
  直近7日間の摂取量を表す、シンプルな手描きアナログ調の縦グラフ。目標達成日はバーが「青（潤い）」になり、未達成日は「茶色（乾き）」、お皿が溢れた日は「サビ色（お皿の漏れ）」で表現。
- **お皿溢れカウント（戒めカウンター）**:
  「今週お皿を溢れさせた回数: 2回」といった、愛嬌のあるペナルティ回数表示。
- **日カレンダー**:
  1か月をグリッドで表示。目標達成日には「小さなかっぱの顔スタンプ」が押され、カレンダーをかっぱで埋めていく楽しみを提供する。

### ④ 設定画面 (Settings View)
ユーザーの体格に合わせたモード調整や、使い勝手を向上させるための調整画面。
- **パーソナル設定**:
  性別モード選択：女性（目標 2500ml） / 男性（目標 3500ml）のトグル切り替え。
- **マイコップ設定（mlカスタム）**:
  ホーム画面のクイック追加ボタン2つの数値を自由にカスタム（例: コップを 200ml に、ボトルを 650ml に設定）。
- **リマインダー（通知）**:
  「1時間ごとに水分補給を促す」「最後の記録から2時間後に通知する」といったスマート通知のON/OFF。
- **データ初期化**:
  アルバムデータのフルリセット、今日の水分の強制リセットボタン（警告ポップアップ付き）。

## 8. 技術的要件
- **ローカル状態保存**: SwiftData、CoreData、または最もシンプルなUserDefaultsを用いて、今日の水分量（ml）、現在の進化ステージ、獲得したかっぱのアルバム（図鑑）データをすべて端末内に永続化。外部APIやOSのヘルスケア機能（HealthKit）との同期処理を一切行わない、完全スタンドアロン構成。
- **フロントエンド**: SwiftUI。
